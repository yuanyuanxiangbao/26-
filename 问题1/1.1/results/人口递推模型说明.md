# 人口递推模型年度操作次序

## 一、模型假设

1. 年均自然死亡率：d = 5%
2. 年均新增老人（刚满60岁）占比：b = 7%
3. 自理→半失能年转移概率：p_s2m = 4.5%
4. 半失能→失能年转移概率：p_m2d = 10%
5. 状态转移不可逆：失能↛半失能↛自理

## 二、年度操作次序

每年按以下三步顺序递推：

```
┌─────────────────────────────────────────────────────────────┐
│                     年度递推流程                              │
└─────────────────────────────────────────────────────────────┘

第t年初状态: S(t), M(t), D(t)
            ↓
┌───────────────────────────────────────┐
│ 步骤1：新增老人（全为自理状态）          │
│                                        │
│ 新增数 = b × [S(t) + M(t) + D(t)]      │
│ S' = S(t) + 新增数                     │
│ M' = M(t)                              │
│ D' = D(t)                              │
└───────────────────────────────────────┘
            ↓
┌───────────────────────────────────────┐
│ 步骤2：全员死亡（含新增老人）            │
│                                        │
│ S'' = S' × (1-d)                       │
│ M'' = M' × (1-d)                       │
│ D'' = D' × (1-d)                       │
└───────────────────────────────────────┘
            ↓
┌───────────────────────────────────────┐
│ 步骤3：状态转移（单向不可逆）            │
│                                        │
│ 自理→半失能: Δ_sm = S'' × p_s2m        │
│ 半失能→失能: Δ_md = M'' × p_m2d        │
│                                        │
│ S(t+1) = S'' - Δ_sm                    │
│ M(t+1) = M'' - Δ_md + Δ_sm             │
│ D(t+1) = D'' + Δ_md                    │
└───────────────────────────────────────┘
            ↓
第t年末（第t+1年初）状态: S(t+1), M(t+1), D(t+1)
```

## 三、状态转移图

```
              b·T(t) 新增（全入自理）
                  │
                  ▼
    ┌────────┐  p_s2m=4.5%  ┌──────────┐  p_m2d=10%  ┌────────┐
    │  自理  │ ────────────→│  半失能  │ ────────────→│  失能  │
    │  S(t)  │              │  M(t)    │              │  D(t)  │
    └────────┘              └──────────┘              └────────┘
         │                        │                        │
         │ d=5%                   │ d=5%                   │ d=5%
         ▼                        ▼                        ▼
       死亡                     死亡                     死亡

约束条件：
- 失能 ↛ 半失能（不可逆）
- 半失能 ↛ 自理（不可逆）
- 新增老人全部为自理状态
```

## 四、递推公式汇总

$$S(t+1) = [S(t) + b \cdot T(t)] \cdot (1-d) \cdot (1-p_{s\to m})$$

$$M(t+1) = M(t) \cdot (1-d) \cdot (1-p_{m\to d}) + [S(t) + b \cdot T(t)] \cdot (1-d) \cdot p_{s\to m}$$

$$D(t+1) = D(t) \cdot (1-d) + M(t) \cdot (1-d) \cdot p_{m\to d}$$

其中 $T(t) = S(t) + M(t) + D(t)$

## 五、数值验证

代入参数 d=0.05, b=0.07：

**年净增长率**：
$$T(t+1) = T(t) \times 1.07 \times 0.95 = T(t) \times 1.0165$$

**5年累计增长**：
$$1.0165^5 - 1 = 8.53\%$$

与代码输出 8.5% 吻合。

## 六、代码实现要点

1. 使用 `round_half_up()` 替代 `round()`，避免银行家舍入
2. 合计 = 自理 + 半失能 + 失能（三个舍入后整数相加）
3. 每个小区独立递推，互不影响
4. 保存每年结果到 `year_0.csv` ~ `year_5.csv`
